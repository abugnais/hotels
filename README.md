# hotels

## Built with
* nodejs@8.5.0
* npm@5.6.0

```bash
sudo apt-get install npm

sudo npm i -g npm@5.6.0

sudo npm i -g nvm

sudo nvm install 8.5.0

npm install

npm start
```

And you can access the app by browsing [http://localhost:3000](http://localhost:3000)
